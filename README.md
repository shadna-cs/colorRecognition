# Color Recognition using OpenCV

## Project Description
This project detects colors in real time using a webcam and OpenCV. It recognizes the following colors:
- Red
- Green
- Blue
- Yellow

When a colored object is detected, the program draws a rectangle around it and displays the color name.

## Requirements
- Python
- OpenCV
- NumPy

Install the required libraries:
pip install -r requirements.txt

## How to Run

Run the following command:
python color_recognition.py

## Project Files

- color_recognition.py : Main program
- requirements.txt : Required libraries
- screenshots : Sample output images
